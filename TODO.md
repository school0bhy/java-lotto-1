1. 로또 생성
    - [x] 1 이상 45 이하 범위의 6개의 중복되지 않는 숫자로 구성 -> 오류 x
    - [x] 중복된 숫자 포함 -> 오류 ㅇ
    - [x] 7개의 숫자로 구성 -> 오류 ㅇ
    - [x] 0 이하 포함 -> 오류 ㅇ
    - [x] 46 이상 포함 -> 오류 ㅇ
    
2. 구입 금액 처리하기
    - [ ] 0보다 작은 구입금액 -> 오류 ㅇ
    - [ ] 1000보다 작은 구입금액 -> 로또 생성 불가
    - [ ] 1000보다 크고, 1000으로 나눈 나머지가 0이 아닌 경우 -> 거스름돈 발생
